**Types and functions to implement in `jsonld.go`:**

```go
package jsonld

// ArticleParams holds the inputs needed to build an Article JSON-LD object.
type ArticleParams struct {
    Title       string
    Description string
    Author      string
    BaseURL     string
    WebPath     string
    SiteName    string
    DateCreated time.Time
    DateModified time.Time
    ImageURL    string  // OG image URL, can be empty
}

// BreadcrumbItem represents one breadcrumb for JSON-LD generation.
type BreadcrumbItem struct {
    Label string
    URL   string
}

// BuildArticleJSON returns a JSON-LD string for schema.org/Article.
// Returns empty string if title is empty.
func BuildArticleJSON(p ArticleParams) string

// BuildBreadcrumbJSON returns a JSON-LD string for schema.org/BreadcrumbList.
// Returns empty string if items is empty.
func BuildBreadcrumbJSON(baseURL string, items []BreadcrumbItem) string
```

**Implementation details:**
- Use `encoding/json` with `json.Marshal` to produce the JSON-LD. Build the schema as `map[string]any` structures.
- `BuildArticleJSON` should produce a `<script type="application/ld+json">` compatible JSON string (just the JSON object, not the script tag — the template will wrap it).
- Article schema: `@context: "https://schema.org"`, `@type: "Article"`, `headline`, `description`, `author: {type: Person, name}`, `datePublished` (RFC3339 from DateCreated), `dateModified` (RFC3339 from DateModified), `publisher: {type: Organization, name: SiteName}`, `mainEntityOfPage: {type: WebPage, @id: BaseURL+WebPath}`, optionally `image` if ImageURL is non-empty.
- If `Author` is empty, omit the `author` field. If `Description` is empty, omit it.
- `BuildBreadcrumbJSON` should produce a `BreadcrumbList` schema with `itemListElement` containing `ListItem` entries with `position` (1-based), `name`, and `item` (full URL = baseURL + item.URL).
- For breadcrumb items where URL is `"#"` (used for current page in tag pages), omit the `item` field from that ListItem.
