# `serve`

You can use the `serve` command to launch a simple web server and preview your generated website.

## Flags

| Flag       | Short version | Default value | Description                                                                             |
| ---------- | ------------- | ------------- | --------------------------------------------------------------------------------------- |
| `--port`   | `-p`          | `"8080"`      | Port to serve on                                                                        |
| `--output` | `-o`          | `"./public"`  | The name of the output directory where the result of the [[generate]] command resides.  |
