# `doctor`

The `doctor` command is used check the health of your vault. Right now it just checks for broken links.

## Flags

| Flag       | Short version | Default value | Description                                  |
| ---------- | ------------- | ------------- | -------------------------------------------- |
| `--input`  | `-i`          | `"vault"`     | Name of the directory containing your vault. |
